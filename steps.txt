}
}

========================
STEP 9: REAL CI/CD (FOR SHOWING)
================================

nano real/real-deploy.yml

name: Real Terraform CI/CD

on:
push:
branches: [ main ]

jobs:
terraform:
runs-on: ubuntu-latest

```
steps:
  - uses: actions/checkout@v3

  - name: Setup Terraform
    uses: hashicorp/setup-terraform@v2

  - name: Terraform Init
    run: |
      cd terraform
      terraform init

  - name: Terraform Apply
    run: |
      cd terraform
      terraform apply -auto-approve
```
